# 📝 To-Do List Web Application

A simple and clean To-Do List web application built using **PHP and MySQL**.  
This project allows users to manage daily tasks easily with basic CRUD features.

---

## 🚀 Features

- Add new tasks 🆕  
- Edit existing tasks ✏️  
- Mark tasks as done ✔️  
- Delete tasks ❌  
- Simple and responsive UI  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Local Server: XAMPP / Laragon
